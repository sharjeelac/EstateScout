**EstateScout** is a modern real estate marketplace application where users can find, list, and filter properties for rent or sale. Built with the MERN Stack (MongoDB, Express, React, Node.js), it features advanced search, image uploads, and authentication.

---

## üöÄ Features (Planned MVP)

- **Authentication:** Secure User & Agent Login (JWT & Bcrypt).
- **Advanced Search:** Filter properties by Location, Price, and Amenities.
- **Property Listings:** Create, Read, Update, and Delete (CRUD) listings.
- **Image Management:** Cloud storage integration using Cloudinary.
- **Responsive UI:** Built with React & Vite for a fast mobile-friendly experience.
